# VR Project

## Setup

Install VS Code. Install Git, install Node JS (make sure `node` and `npm` are in your PATH). 

```
git clone https://github.com/keshav1/VR-Project.git
cd VR-Project
npm install
npm install -g serve
```

## Run project

```
serve
```

Open http://localhost:5000 in your browser. Yay!
